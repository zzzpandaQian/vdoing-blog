# VUE CLI安装

CLI依赖于Node所以要先安装nodejs

## 一.CLI3: npm install -g @vue/cli

## 二.CLI2: npm install @vue/cli-init -g

## 三.项目创建

### 1.CLI2: vue init webpack 项目名称

### 2.CLI3 ： vue create 项目名称

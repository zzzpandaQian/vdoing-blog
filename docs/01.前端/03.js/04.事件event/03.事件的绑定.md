# 事件的绑定

## 1.addEventListener('eventname', function, true/false);

### 其中eventname指代click,mouseover,mousedown等，注意不能加on

### true和false指代是否遵循非冒泡事件顺序即由内向外响应。

### 通常填写false（必填）即默认响应顺序。
